Simple Online Duel

How to run:
1. Open a terminal in this folder
2. Run: npm install
3. Run: npm start
4. Open http://localhost:3000

How to play online:
- Player 1 clicks "Create Room"
- Player 2 opens the same website and clicks "Join Room"
- Share the 4-letter room code

For internet play:
- Deploy this folder to Render, Railway, or any Node host
- Then both players open the deployed URL

Notes:
- This is intentionally the easiest simple version
- It uses room codes and live browser sync
- It is server-based, so both players stay in sync
